# Documentation

This directory contains Leamas project documentation.

## Structure

- [doctrine/](doctrine/) — Core principles and design philosophy
- [adr/](adr/) — Architecture Decision Records
- [playbooks/](playbooks/) — Runbooks and operational playbooks
- [install/](install/) — Installation guides
- [acts/](acts/) — Active change contracts
- [templates/](templates/) — Document templates for epics, acts, etc.
- [close-reports/](close-reports/) — Close reports for completed ACTs and Epics

## Sections

### Doctrine

Core principles that guide Leamas development decisions.

### ADR

Architecture Decision Records capturing significant technical choices.

### Playbooks

Operational runbooks for common tasks and procedures.

### Templates

Document templates used in the project workflow.

### Close Reports

Close reports capture completed ACT and Epic outcomes with verification evidence.
