# Close Reports

Close reports capture the completed outcomes of ACTs and Epics. They document what was done, how it was verified, and what follows.

## Purpose

When an ACT or Epic is closed, a close report should be created to:
- Document the concrete deliverables
- Record verification evidence
- Track reviewer feedback incorporation
- List follow-up work
